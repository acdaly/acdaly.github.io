OVERVIEW


As a LearnSphere intern, I worked to redesign and create designs from scratch for their projects Tigris and PL². Tigris allows educational researchers and educators to use and create existing or custom data analysis workflows. As an example, uploading a gradebook to Tigris could allow a user to determine which activities/quizes/tests were the best predictors of students' final greades. PL² on the otherhand is a research project addressing the opportunity gap for marginalized students through personal mentoring and tutoring with artificial intelligence learning software.

For Tigris, I conducted user interviews, redesigned the main page of the tool, and created a style guide for the continued use of the team's current developers and future designers. For the PL² web app, I redesigned the home page, created designs for the new Administrator features, and collaborated with the other design intern, Patricia Yu, to create a consistent style guide. I additionally both designed and coded the new PL² website from scratch.
