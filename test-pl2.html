The Challenge

As a product design intern at LearnLab, I was tasked to redesign parts of Personalized Learning² (PL²) to make it more intuitive for first-time users.

PL² is a project addressing the opportunity gap for marginalized students through personal mentoring and tutoring with artificial intelligence learning software. There was a low conversion rate of mentors in partnered schools using the PL² web app, so I was tasked with improving the app’s usability.

Scope

I had 8 weeks* to redesign PL² while working on another LearnLab project.

I collaborated with Patricia Yu (UI/UX design intern) to create the design system, and worked with Cindy Tipper (our project manager and developer), 3 other developers, and 2 learning scientists on the project.

My tasks were to:

    1.    Redesign key features of PL² (home page & admin page).
    2.    Design and develop a new informational website for PL².


*During my 8-week internship, I balanced simultaneously working on another LearnLab project, Tigris.

PL² Design

Creating a Distinct Design System


I collaborated with Patricia Yu (design intern) to update the design system used for PL². We wanted to give PL² a distinct, professional style. The more muted colors made it easier to draw attention with our pops of blue, green, and red.

We updated the typography, colors, and several design components and documented these changes in a style guide passed to the developers on the team.

PL² Home Page

Meeting Admin and Mentor Needs with an Improved Visual System and Sorting Options

The new home page has visual cues for the clickable student names, improved hierarchy of text, use of the grid, and sorting options.

PL² Admin page

Empowering Admins to Analyze Data with an Intuitive Secondary Navigation and Updated User Flow

The secondary navigation gives a clear overview of the key features school administrators have access to. The list and graph views allows program administrators oversight of the mentors and students’ progress.

PL² Landing page

An Inclusive, Illustrated, and Informative Introduction to PL²

I designed, developed, and illustrated an informative website to allow prospective administrators to learn more about the project, while also giving current users a more welcoming landing page.

View the site at https://www.cs.cmu.edu/~pl2/

reflection

Collaborating with Designers and Developers

This internship was the first time I created a style guide for a live product. Collaborating with another designer on the design system made me really think about why I had chosen the exact hex code I did. The design system overall helped me simplify the design. I also learned how I could communicate my design in a way a developer could implement. I realized that my design would need more in depth notes to highlight decisions I took for granted (adhering to the grid system, creating symmetrical margins, etc.).
